This plugin allows you to easily add the Orimon chatbot to your WordPress site.

== Description ==

Orimon Chatbot is a plugin that allows you to easily integrate the Orimon chatbot into your WordPress site. With this plugin, you can provide your website visitors with a seamless customer service experience by enabling them to get their questions answered through a chatbot powered by ChatGPT style artificial intelligence.

Once installed and activated, you can customize the chatbot settings by entering your Orimon Chatbot ID.

To get started, simply install and activate the plugin, then go to the Orimon Chatbot settings page under the WordPress Settings menu to configure the plugin.

== Installation ==

Upload the plugin files to the /wp-content/plugins/orimon-chatbot directory, or install the plugin through the WordPress plugins screen directly.
Activate the plugin through the 'Plugins' screen in WordPress
Go to the Orimon Chatbot settings page under the WordPress Settings menu to configure the plugin.
== Frequently Asked Questions ==

= How do I get my Orimon Chatbot ID? =

To get your Orimon Chatbot ID, please visit https://orimon.ai/ and follow the instructions provided on the platform.

= Can I customize the chatbot appearance? =

Yes, you can customize the appearance of the chatbot by adding CSS styles to your WordPress theme.
